# 历史模块设计

## 一、设计目标

历史模块负责：
1. **历史记录**：记录会话的所有消息和事件
2. **历史回放**：支持会话历史的回放
3. **历史导出**：支持导出历史记录

## 四、回放服务实现

```python
class HistoryPlayback:
    """历史回放服务"""
    
    def __init__(self, history_service: IHistoryService):
        self.history_service = history_service
        self._playbacks: Dict[str, PlaybackSession] = {}
    
    async def create_playback(
        self,
        session_id: str
    ) -> PlaybackSession:
        """创建回放会话"""
        messages = await self.history_service.get_history(session_id)
        
        playback = PlaybackSession(
            id=f"playback_{uuid.uuid4().hex[:8]}",
            session_id=session_id,
            events=[HistoryEvent.from_message(m) for m in messages],
            current_index=0
        )
        
        self._playbacks[playback.id] = playback
        return playback
    
    async def get_next_event(
        self,
        playback_id: str
    ) -> Optional[HistoryEvent]:
        """获取下一事件"""
        playback = self._playbacks.get(playback_id)
        if not playback:
            return None
        
        if playback.current_index >= len(playback.events):
            return None
        
        event = playback.events[playback.current_index]
        playback.current_index += 1
        return event
    
    async def seek_to(
        self,
        playback_id: str,
        index: int
    ) -> None:
        """跳转到指定位置"""
        playback = self._playbacks.get(playback_id)
        if playback:
            playback.current_index = max(0, min(index, len(playback.events) - 1))
```
