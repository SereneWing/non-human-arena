# 依赖注入容器设计

## 一、设计目标

依赖注入容器是系统核心的组件管理模块，负责：
1. **管理组件生命周期**：创建、初始化、销毁组件
2. **解决依赖关系**：自动解析和注入组件依赖
3. **统一组件获取**：提供统一的组件访问接口
4. **支持多种作用域**：Singleton、Transient、Scoped

## 二、设计原则

1. **延迟初始化**：按需创建组件，避免启动时创建所有组件
2. **循环依赖检测**：检测并报告循环依赖
3. **接口抽象**：依赖抽象而非具体实现
4. **可配置性**：支持自定义依赖绑定

## 四、核心接口

```python
from typing import Type, TypeVar, Callable, Optional, Dict, Any
from enum import Enum

T = TypeVar('T')

class Scope(Enum):
    """生命周期作用域"""
    SINGLETON = "singleton"      # 单例：整个应用生命周期
    TRANSIENT = "transient"     # 临时：每次获取都创建新实例
    SCOPED = "scoped"          # 作用域：每个请求/会话创建一个实例

class IContainer(Protocol):
    """容器接口"""
    
    def register(
        self,
        abstract: Type[T],
        concrete: Type[T] = None,
        scope: Scope = Scope.SINGLETON
    ) -> None:
        """注册组件"""
        ...
    
    def register_instance(
        self,
        abstract: Type[T],
        instance: T
    ) -> None:
        """注册实例"""
        ...
    
    def register_factory(
        self,
        abstract: Type[T],
        factory: Callable[..., T]
    ) -> None:
        """注册工厂函数"""
        ...
    
    async def resolve(self, abstract: Type[T]) -> T:
        """解析组件"""
        ...
    
    def create_scope(self) -> "IScope":
        """创建作用域"""
        ...
```

## 五、使用示例

```python
# 基本用法
container = Container()

# 注册服务
container.register(IRoleRepository, RoleRepository, Scope.SINGLETON)
container.register(IRuleEngine, RuleEngine, Scope.SINGLETON)

# 注册服务实例
event_bus = LocalEventBus()
container.register_instance(IEventBus, event_bus)

# 注册工厂函数
container.register_factory(
    ILLMAdapter,
    lambda config: OpenAIAdapter(config)
)

# 解析服务
role_repo = await container.resolve(IRoleRepository)
```

## 六、生命周期管理

### 6.1 Singleton（单例）
- 应用启动时注册
- 整个应用生命周期只创建一个实例
- 适合无状态服务

### 6.2 Transient（临时）
- 每次解析都创建新实例
- 适合有状态或轻量级组件

### 6.3 Scoped（作用域）
- 每个作用域创建一个实例
- 适合Web请求、会话等场景
- 需要配合作用域管理

## 七、与事件总线的集成

```python
class ContainerEventHandler:
    """容器事件处理器"""
    
    def __init__(self, container: IContainer):
        self.container = container
    
    @on_event(EventType.SYSTEM_STARTUP)
    async def on_startup(self, event: Event):
        """系统启动时初始化关键服务"""
        # 预热关键服务
        await self.container.resolve(IRoleService)
        await self.container.resolve(IRuleService)
    
    @on_event(EventType.SYSTEM_SHUTDOWN)
    async def on_shutdown(self, event: Event):
        """系统关闭时清理资源"""
        if hasattr(self.container, 'dispose'):
            await self.container.dispose()
```
