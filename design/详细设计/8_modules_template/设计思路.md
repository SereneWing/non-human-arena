# 模板模块设计

## 一、设计目标

模板模块负责：
1. **模板管理**：角色、规则、场景模板的创建和管理
2. **模板渲染**：将变量替换为实际值
3. **模板继承**：支持模板继承和覆盖

## 四、模板渲染器实现

```python
import re
from typing import Dict, Any

class TemplateRenderer:
    """模板渲染器"""
    
    VARIABLE_PATTERN = re.compile(r'\{\{(\w+)\}\}')
    
    async def render(
        self,
        template: Template,
        variables: Dict[str, Any]
    ) -> Dict[str, Any]:
        """渲染模板"""
        result = {}
        
        for key, value in template.content.items():
            if isinstance(value, str):
                result[key] = await self.render_string(value, variables)
            elif isinstance(value, dict):
                result[key] = await self._render_dict(value, variables)
            elif isinstance(value, list):
                result[key] = await self._render_list(value, variables)
            else:
                result[key] = value
        
        return result
    
    async def render_string(
        self,
        text: str,
        variables: Dict[str, Any]
    ) -> str:
        """渲染字符串模板"""
        def replace_var(match):
            var_name = match.group(1)
            return str(variables.get(var_name, match.group(0)))
        
        return self.VARIABLE_PATTERN.sub(replace_var, text)
    
    async def _render_dict(
        self,
        obj: Dict[str, Any],
        variables: Dict[str, Any]
    ) -> Dict[str, Any]:
        """渲染字典"""
        result = {}
        for key, value in obj.items():
            if isinstance(value, str):
                result[key] = await self.render_string(value, variables)
            elif isinstance(value, dict):
                result[key] = await self._render_dict(value, variables)
            else:
                result[key] = value
        return result
    
    async def _render_list(
        self,
        items: List[Any],
        variables: Dict[str, Any]
    ) -> List[Any]:
        """渲染列表"""
        result = []
        for item in items:
            if isinstance(item, str):
                result.append(await self.render_string(item, variables))
            elif isinstance(item, dict):
                result.append(await self._render_dict(item, variables))
            else:
                result.append(item)
        return result
```
