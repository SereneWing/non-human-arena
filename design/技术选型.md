# 技术选型

## 一、技术栈总览

| 层级 | 技术 | 版本 | 用途 |
|------|------|------|------|
| **后端框架** | FastAPI | 0.100+ | API服务、异步处理 |
| **数据库** | SQLite (本地) / PostgreSQL (服务端) | - | 持久化存储 |
| **缓存** | Redis (服务端) / 内存 (本地) | - | 会话缓存、实时通信 |
| **消息队列** | Redis Pub/Sub (本地) / RabbitMQ (服务端) | - | 事件总线 |
| **ORM** | SQLAlchemy | 2.0+ | 数据库操作 |
| **前端框架** | Vue.js | 3.4+ | 前端界面 |
| **状态管理** | Pinia | 2.1+ | 状态管理 |
| **UI组件** | Element Plus | 2.4+ | UI组件库 |
| **构建工具** | Vite | 5.0+ | 前端构建 |
| **类型检查** | TypeScript | 5.0+ | 类型安全 |
| **容器化** | Docker | 24.0+ | 部署 |

## 二、后端技术选型

### 2.1 Web框架：FastAPI

**选型理由**：
- 异步优先，性能优秀
- 自动生成OpenAPI文档
- 类型注解支持，与Pydantic无缝集成
- 依赖注入系统成熟
- Starlette基础，稳定可靠

**备选方案**：
| 方案 | 优点 | 缺点 |
|------|------|------|
| Flask | 轻量、灵活 | 异步支持弱 |
| Django | 功能全面 | 重量级、异步复杂 |
| Sanic | 高性能 | 生态较小 |

### 2.2 数据库：SQLAlchemy 2.0

**选型理由**：
- 异步支持完善
- 类型注解丰富
- SQLAlchemy 2.0改进巨大
- 支持多种数据库后端

**本地模式**：
- SQLite：零配置、文件存储、便携

**服务端模式**：
- PostgreSQL：可靠性高、功能丰富

### 2.3 数据验证：Pydantic v2

**选型理由**：
- FastAPI原生集成
- 类型验证强大
- 自动生成JSON Schema
- 性能优秀

### 2.4 异步任务：asyncio + 内存队列

**本地模式**：
- 直接使用asyncio
- 无需额外依赖

**服务端模式**：
- Redis：成熟稳定
- Celery（可选）：任务队列
- RabbitMQ（可选）：高吞吐量

## 四、通信技术选型

- CRUD操作

- HTTP/1.1（兼容好）

**用途**：

## 八、安全考虑

### 8.1 本地模式安全

- 无网络暴露
- 数据完全本地
- 无认证需求

### 8.2 服务端模式安全

| 安全措施 | 实现方式 |
|----------|----------|
| 认证 | JWT Token |
| 授权 | RBAC角色权限 |
| 限流 | Redis计数器 |
| 加密 | HTTPS + TLS |
| 审计 | 日志记录 |

### 8.3 API安全中间件

```python
class SecurityMiddleware:
    def __init__(self):
        self.rate_limiter = RateLimiter(
            requests_per_minute=60,
            burst=10
        )
        self.auth = JWTAuth(secret_key="...")
    
    async def process(self, request, call_next):
        # 1. 认证检查
        await self.auth.verify(request)
        
        # 2. 限流检查
        await self.rate_limiter.check(request)
        
        # 3. 处理请求
        return await call_next(request)
```

## 九、性能优化

### 9.1 后端优化

- **异步处理**：所有IO操作异步化
- **连接池**：数据库、Redis连接池
- **缓存**：热点数据缓存
- **限流**：防止资源耗尽

### 9.2 前端优化

- **懒加载**：路由和组件懒加载
- **虚拟滚动**：长列表优化
- **防抖节流**：用户输入处理
- **Tree Shaking**：减少包体积

### 9.3 预估性能指标

| 指标 | 本地模式 | 服务端模式 |
|------|----------|------------|
| API响应时间 | < 100ms | < 200ms |
| WebSocket延迟 | < 50ms | < 100ms |
| 并发会话 | 10+ | 100+ |
| 同时连接 | 50+ | 500+ |
