# 数据结构介绍

## 一、概述

本目录包含系统中所有核心数据结构的定义。这些数据结构用于：
1. 定义角色模板
2. 定义规则模板
3. 管理会话状态
4. 定义AI实体
5. 消息传递

## 二、数据结构分类

| 类别 | 数据结构 | 说明 |
|------|----------|------|
| **角色** | Role, RoleTemplate, MentalConfig, EmotionalModel, SpeakingStyle | AI角色定义 |
| **规则** | Rule, RuleTemplate, Constraint, Trigger, Consequence | 讨论规则定义 |
| **会话** | Session, Participant, Message, SessionState | 会话管理 |
| **AI** | AIAgent, AIDecision, MentalActivity, Skill | AI行为定义 |
| **模板** | SceneTemplate | 场景模板 |

## 三、设计原则

### 3.1 标准化
- 所有数据结构使用标准JSON可序列化格式
- 字段命名采用snake_case
- 枚举值采用kebab-case

### 3.2 扩展性
- 使用@dataclass支持继承和扩展
- 支持模板变量替换
- 支持默认值

### 3.3 验证性
- 使用Pydantic进行数据验证
- 字段类型注解明确
- 支持自定义验证器

## 七、使用示例

### 7.1 创建角色
```python
from app.modules.role.schemas import Role, MentalConfig, EmotionalModel

role = Role(
    id="role_001",
    name="正方辩手",
    description="支持辩题的辩手",
    base_personality={"logical": 0.9, "aggressive": 0.6},
    mental_config=MentalConfig(
        skip_probability=0.2,
        enable_mental_activity=True
    ),
    emotional_model=EmotionalModel(
        dimensions=["confidence", "determination"]
    )
)
```

### 7.2 创建规则
```python
from app.modules.rule.schemas import RuleTemplate, Constraint, Trigger

rule = RuleTemplate(
    id="debate.standard",
    name="标准辩论",
    constraints=[
        Constraint(
            type="message_length",
            target="participant",
            condition="len <= 500",
            error_message="发言不能超过500字"
        )
    ],
    triggers=[
        Trigger(
            event_type="message_received",
            condition="true"
        )
    ]
)
```

### 7.3 创建会话
```python
from app.modules.session.schemas import Session, Participant

session = Session(
    id="session_001",
    topic="AI是否应该取代人类工作",
    participants=[
        Participant(role_id="role_001", name="正方"),
        Participant(role_id="role_002", name="反方")
    ],
    state="created"
)
```
